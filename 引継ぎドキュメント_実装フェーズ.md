## Store 設計

### useSettingsStore
```typescript
state: {
  language: 'ja' | 'en'
  selectedLevels: number[]
  quizCount: 5 | 10 | 20
  isOnboardingDone: boolean
}
// setLanguage → i18next.changeLanguage() を同時実行
// setSelectedLevels → wordStore.updateFilteredWords() を呼ぶ
// resetSettings → 全Store初期化
```

### useWordStore
```typescript
state: {
  allWords: Word[]
  filteredWords: Word[]      // selectedLevels で絞込済み
  knownWordIds: Set<string>
}
// computed: knownCount / totalCount
```

### useTestStore
```typescript
state: {
  questions: TestQuestion[]
  currentIndex: number
  answers: boolean[]
  mode: 'test' | 'retest'
}
// computed: score / scorePercent / wrongWords
```

### useMistakeStore
```typescript
state: {
  records: Record<string, MistakeRecord>
}
// 3回連続正解で isMastered = true
// computed: unmastered（ホームバッジ用）
```

## 4択ダミー生成ロジック（コスト¥0）

```typescript
const generateChoices = (correct: Word, allWords: Word[], lang: 'ja' | 'en') => {
  const correctMeaning = lang === 'ja' ? correct.meaning_ja : correct.meaning_en;
  const dummies = allWords
    .filter(w => w.id !== correct.id && w.level === correct.level)
    .sort(() => Math.random() - 0.5)
    .slice(0, 3)
    .map(w => lang === 'ja' ? w.meaning_ja : w.meaning_en);
  return [correctMeaning, ...dummies].sort(() => Math.random() - 0.5);
};
```

## UI 特記事項

- テスト正解時: 緑帯フィードバック → 0.8秒後に自動遷移
- テスト不正解時: 赤帯フィードバック → 0.8秒後に自動遷移
- 満点（10/10）時: 金バッジ＋アニメーション（ぷるぷる）
- 誤答バッジ色: 赤（3回以上）/ 橙（2回）/ 緑（1回）
- タブレット対応: `useDeviceType()` で幅768px以上を判定

## コーディング規約

- コンポーネントは関数型のみ
- props の型定義は必ず書く（`interface Props {...}`）
- `any` 型禁止
- UI テキストは必ず `t('key')` 経由
- スタイルは各ファイル末尾の `StyleSheet.create()` にまとめる
- APIキーのハードコード禁止（MVP では外部API不使用のため不要）
